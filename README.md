## About Project

The Story Books is a project developed to learn and apply Node.js technologies. Its primary purpose is to provide users with a platform to create their stories and share with others.

## Used Technologies

- Nodejs / Expressjs
- Mongodb / Mongoose
- Passportjs
- Google OAuth
- Sessions
- Handlebars
- MVC pattern
